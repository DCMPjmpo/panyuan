# 诉讼策略建议

## 目的
全面分析案件的优劣势、关键证据、法律依据和对手可能的应对，制定最优的诉讼策略（包括是否起诉、何时起诉、如何起诉）。

## 何时使用
- 用户考虑起诉但不确定胜算时
- 对方已起诉需要制定应诉策略时
- 触发词：诉讼策略、打官司策略、起诉还是和解、诉讼分析、案件评估、怎么打

## 工作流程
1. **事实分析**：
   - 梳理有利事实和不利事实
   - 识别关键争议焦点
2. **法律分析**：
   - 确定适用的法律法规
   - 分析请求权基础是否充分
   - 识别法律上的薄弱环节
3. **证据分析**：
   - 现有证据的证明力评估
   - 缺失证据的影响
   - 对方可能持有的反证
4. **对手分析**：预测对方可能的抗辩理由和反击策略
5. **诉讼时机分析**：考虑诉讼时效、管辖权、财产保全时机
6. **和解评估**：比较诉讼成本和预期收益，评估和解方案的合理性
7. **策略建议**：给出具体的行动方案（起诉/调解/和解/暂缓）

## 输出格式
- **案件SWOT分析**：优势/劣势/机会/威胁
- **关键争议焦点**：决定胜负的核心问题
- **胜诉率评估**：分乐观/中性/悲观三种场景
- **诉讼成本估算**：诉讼费+律师费+时间成本
- **和解区间建议**：合理的和解金额范围
- **行动方案**：按优先级排列的策略建议
