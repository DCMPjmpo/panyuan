# 仲裁申请书

## 目的
根据仲裁协议的约定，生成符合仲裁委员会要求的仲裁申请书，包括仲裁依据、申请事项、事实理由和证据目录。

## 何时使用
- 合同约定了仲裁条款，需要申请仲裁时
- 当事人协商一致选择仲裁解决纠纷时
- 触发词：仲裁申请、商事仲裁、仲裁委员会、申请仲裁、仲裁协议、提交仲裁

## 工作流程
1. **确认仲裁依据**：检查合同中是否有有效的仲裁条款或另行达成的仲裁协议
2. **确定仲裁机构**：根据仲裁条款确定具体的仲裁委员会
3. **收集当事人信息**：申请人和被申请人的基本信息
4. **明确仲裁请求**：梳理具体的仲裁请求事项
5. **计算仲裁费用**：根据仲裁规则估算仲裁费用
6. **梳理事实与理由**：整理纠纷事实，引用合同条款和法律规定
7. **准备证据材料**：列出需要提交的证据清单
8. **生成仲裁申请书**：按照仲裁委格式要求生成完整文书

## 输出格式
- **标题**：仲裁申请书
- **申请人信息**
- **被申请人信息**
- **仲裁依据**：仲裁条款或仲裁协议
- **仲裁请求**：分条列出具体请求
- **事实与理由**：纠纷经过和法律依据
- **此致/仲裁委员会**
- **申请人/日期**
- **证据清单**
