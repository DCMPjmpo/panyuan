# 答辩状生成

## 目的
根据原告的起诉状内容，生成民事答辩状，逐条回应原告诉讼请求，提出反驳事实和法律依据。

## 何时使用
- 用户收到起诉状需要答辩时
- 用户被起诉需要准备应诉材料时
- 触发词：答辩状、被起诉、收到传票、被告、如何答辩、反驳起诉、应诉

## 工作流程
1. **分析起诉状**：梳理原告的诉讼请求和事实理由
2. **逐条回应**：对原告的每项诉讼请求和每段事实主张逐一确认或否认
3. **提出反驳事实**：提供对被告有利的事实版本和证据
4. **引用法律依据**：引用支持被告立场的法律法规
5. **提出管辖异议**（如有）：检查是否可以向其他法院移送
6. **提出反诉**（如有）：分析是否有可以反诉的事由
7. **生成答辩状**：按照法院格式生成完整答辩文书

## 输出格式
按照民事答辩状标准格式：
- **标题**：民事答辩状
- **答辩人信息**
- **答辩意见**：对诉讼请求的整体态度
- **事实与理由**：逐条回应原告主张
- **证据清单**：答辩人提交的证据
- **此致/管辖法院**
- **答辩人/日期**
