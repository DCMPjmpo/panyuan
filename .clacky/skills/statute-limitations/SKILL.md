# 诉讼时效提醒

## 目的
根据权利类型和事实发生时间，计算诉讼时效的起算点、届满日期，提醒用户及时行使权利，避免因时效过期丧失胜诉权。

## 何时使用
- 用户询问某项权利是否已过诉讼时效时
- 用户需要确认维权的最后期限时
- 触发词：诉讼时效、时效、过期没有、还能告吗、多久有效、追诉时效、是不是太晚了

## 工作流程
1. **识别权利类型**：合同债权/侵权赔偿/劳动争议/刑事追诉等
2. **确定时效期间**：根据权利类型确定法定时效长度
3. **确定起算点**：时效从何时开始计算（知道或应当知道权利受损之日）
4. **判断时效中断/中止**：分析是否存在使时效中断或中止的事由
5. **计算届满日期**：精确计算时效的最后一天
6. **紧急程度评估**：距时效届满还有多久，是否需要紧急行动

## 各权利类型的时效一览
- **普通诉讼时效**：3年（民法典第188条）
- **劳动争议仲裁时效**：1年（从知道或应当知道权利被侵害之日起）
- **行政诉讼时效**：6个月（从知道行政行为之日起）
- **刑事追诉时效**：根据法定最高刑期不同，5-20年不等
- **执行时效**：2年（从法律文书生效之日起）
- **撤销权时效**：1年（从知道撤销事由之日起）

## 输出格式
- **权利类型**和适用时效
- **时效计算**：
  - 起算点：XXXX年XX月XX日
  - 时效期间：X年
  - 届满日期：XXXX年XX月XX日
  - 剩余时间：XX个月/天
- **时效中断/中止分析**
- **紧急程度评估**：🟢充足 / 🟡紧张 / 🔴即将到期 / ⚫已过期
- **行动建议**：是否需要立即采取法律行动
