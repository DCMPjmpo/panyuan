# 起诉状生成

## 目的
根据用户提供的案件事实，自动生成符合法院格式要求的民事起诉状，包含诉讼请求、事实与理由、证据清单。

## 何时使用
- 用户需要起草起诉状向法院起诉时
- 用户描述纠纷事实后需要形成法律文书时
- 触发词：起诉状、起诉、打官司、告谁、民事起诉、诉讼请求、写起诉书、我要起诉

## 工作流程
1. **收集信息**：询问原告和被告的基本信息（姓名/名称、住所、联系方式）
2. **确认诉讼请求**：帮用户梳理具体诉求（如：要求支付货款XX元、返还财产、赔偿损失等）
3. **梳理事实经过**：按时间顺序整理纠纷发生的经过
4. **确定法律依据**：根据案件性质，引用相关法条
5. **计算诉讼费用**：根据诉讼标的额，估算案件受理费
6. **确认管辖法院**：根据被告住所地或合同履行地确定有管辖权的法院
7. **生成起诉状**：按照标准格式（首部、诉讼请求、事实与理由、尾部）生成完整文书
8. **附证据清单**：列出需要提交的证据材料

## 输出格式
按照民事起诉状标准格式：
- **标题**：民事起诉状
- **当事人信息**：原告和被告的详细信息
- **诉讼请求**：分条列出具体诉求
- **事实与理由**：按时间线叙述事实，引用法律依据
- **此致/管辖法院**
- **具状人/日期**
- **证据清单**：编号、证据名称、证明目的
