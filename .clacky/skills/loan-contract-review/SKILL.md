# 借款合同审查

## 目的
审查借款合同的利率是否合法、担保是否充分、还款计划是否合理、违约责任是否适当，防范借贷法律风险。

## 何时使用
- 用户提供借款合同需要审查时
- 用户需要了解民间借贷的合法利率上限
- 触发词：借款合同、借贷协议、民间借贷、贷款合同、借条、欠条、借款纠纷

## 工作流程
1. **确认借贷类型**：民间借贷/金融机构贷款/企业间借贷，适用不同法律规则
2. **审查主体**：出借人和借款人的身份信息和履约能力
3. **审查利率条款**：对照LPR四倍标准判断利率是否合法（民间借贷司法保护上限）
4. **审查借款金额与期限**：本金是否明确、还款期限是否清晰
5. **审查担保条款**：保证人资格、抵押物描述、质押手续是否完备
6. **审查还款方式**：等额本息/先息后本/一次性还本付息
7. **审查违约责任**：逾期利率、违约金、复利是否超过法定上限
8. **审查争议解决**：管辖约定是否有效（民间借贷可在原告所在地起诉）
9. **出具审查意见**：标注违法条款，建议修改方向

## 输出格式
- **借贷基本信息**：借贷双方、金额、利率、期限
- **利率合规性**：是否超过LPR四倍/年利率24%/36%标准
- **担保评估**：担保方式的合法性与充分性
- **风险等级**：低/中/高
- **修改建议**：具体条款的合规化建议
