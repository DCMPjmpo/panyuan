# 知识产权保护

## 目的
为企业和个人提供专利、商标、著作权、商业秘密的综合保护策略，包括确权、维权和侵权应对方案。

## 何时使用
- 用户需要保护自己的知识产权时
- 用户遭遇知识产权侵权需要应对方案时
- 触发词：知识产权保护、专利保护、商标保护、著作权、IP策略、知识产权侵权、盗版、抄袭

## 工作流程
1. **识别知识产权类型**：专利（发明/实用新型/外观设计）/商标/著作权/商业秘密/集成电路布图设计
2. **确权建议**：
   - 专利：分析可专利性，建议申请类型和策略
   - 商标：分析可注册性，建议类别选择和防御性注册
   - 著作权：说明自动保护原则和自愿登记的价值
3. **侵权分析**：
   - 判断是否构成侵权（对比分析）
   - 收集侵权证据的方法
   - 评估侵权损害范围
4. **维权路径**：
   - 发律师函/警告函
   - 行政投诉（向知识产权局/市场监管局/版权局举报）
   - 平台投诉（电商平台/社交平台的知识产权投诉渠道）
   - 民事诉讼（诉前禁令+损害赔偿）
   - 刑事报案（严重侵权案件）
5. **输出保护方案**：包含时间线、成本估算和成功率评估

## 输出格式
- **知识产权梳理**：用户拥有的知识产权清单
- **保护策略**：确权/维权/管理三位一体的保护方案
- **侵权应对流程图**：从发现侵权到最终解决的路径图
- **时间与成本估算**：各保护措施的时间线和费用
- **风险提示**：保护漏洞和应对策略
