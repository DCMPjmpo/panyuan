# 技术合同审查

## 目的
审查技术开发、技术转让、技术许可合同中的知识产权归属、技术验收标准、后续改进成果的分配等核心条款。

## 何时使用
- 用户需要审查技术开发/转让/许可合同时
- 用户对技术成果的知识产权归属有疑问时
- 触发词：技术合同、技术开发、技术转让、技术许可、软件合同、源代码、知识产权归属

## 工作流程
1. **确认合同类型**：技术开发（委托/合作）/技术转让/技术许可，不同类错误适用不同规则
2. **审查技术描述**：技术内容、技术指标、技术文档是否完整明确
3. **审查知识产权归属**：现有知识产权、开发过程中产生的知识产权、后续改进的归属
4. **审查验收标准**：技术验收的具体指标、测试方法、验收期限
5. **审查付款条款**：付款是否与技术里程碑挂钩
6. **审查保密条款**：技术秘密的保护措施、保密期限
7. **审查侵权担保**：技术提供方是否保证不侵犯第三方知识产权
8. **审查后续改进**：后续改进成果的使用权和所有权分配
9. **出具审查意见**：标注知识产权风险，建议修改方向

## 输出格式
- **合同类型**与技术概要
- **知识产权归属分析**：各阶段成果的所有权与使用权
- **风险等级**：低/中/高
- **重点风险提示**：知识产权归属不清、验收标准模糊
- **修改建议**：具体条款的改进措辞
