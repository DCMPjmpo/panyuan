# 股权架构设计

## 目的
分析创业公司的股权架构需求，综合考虑创始人控制权、团队激励、融资需求和税务优化，提供个性化股权架构方案。

## 何时使用
- 创业团队需要设计初始股权架构时
- 公司准备融资需要调整股权结构时
- 触发词：股权架构、股权设计、股权分配、合伙人股权、股权比例、AB股、持股平台

## 工作流程
1. **了解创业团队**：创始人数量、各自贡献（资金/技术/资源）、全职/兼职状态
2. **确认控制权需求**：创始人是否需要绝对控制权（>50%或>2/3）
3. **预留员工期权池**：建议预留10-20%用于员工股权激励
4. **设计股权成熟机制**：设置vesting计划（通常4年+1年cliff）
5. **考虑持股方式**：自然人直接持股/有限合伙持股平台/有限公司持股
6. **税务优化分析**：不同持股方式的税收差异（股息、转让）
7. **融资稀释规划**：模拟天使轮~A轮融资后的股权稀释
8. **输出架构方案**：包含股权结构图和控制权分析

## 输出格式
- **推荐股权架构**：各主体持股比例和持股方式
- **股权结构图**：文字描述的树状结构
- **控制权分析**：股东会表决权分布
- **股权成熟计划**：vesting时间表和条件
- **期权池设置建议**：规模、行权价格、行权条件
- **税务提示**：不同持股方式的税务比较
- **融资稀释预测**：各轮次后的股权变化
- **风险提示**：股权架构的法律和税务风险
