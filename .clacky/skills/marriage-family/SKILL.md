# 婚姻家事

## 目的
解答婚姻关系中的法律问题，包括结婚条件、夫妻财产制度、离婚程序与条件、财产分割、子女抚养权、抚养费和家庭暴力保护。

## 何时使用
- 用户咨询离婚、财产分割、子女抚养问题时
- 遭遇家庭暴力需要法律保护时
- 触发词：离婚、财产分割、子女抚养、结婚、彩礼、家庭暴力、抚养费、出轨、净身出户

## 工作流程
1. **确认具体问题**：离婚/财产/子女/家暴/彩礼/其他
2. **离婚问题**：
   - 协议离婚：条件、冷静期（30天）、所需材料
   - 诉讼离婚：法定离婚情形（感情破裂的认定标准）
3. **财产分割**：
   - 区分婚前/婚后财产
   - 夫妻共同财产的范围与分割原则
   - 房产分割的特殊规则
4. **子女抚养**：
   - 抚养权归属的判断标准
   - 抚养费的计算标准
   - 探视权的安排
5. **家庭暴力**：人身安全保护令的申请条件和流程
6. **彩礼问题**：彩礼返还的法定情形
7. **实务建议**：证据收集和维权路径

## 输出格式
- **问题定性**：属于哪类婚姻家事纠纷
- **法律规定**：民法典婚姻家庭编的相关条文
- **实务分析**：结合具体情况的详细分析
- **证据建议**：需要收集的关键证据
- **维权路径**：协商→调解→诉讼的路径
- **心理支持资源**：家暴受害人可求助的机构
