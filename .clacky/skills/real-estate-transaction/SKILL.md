# 房产交易

## 目的
指导房屋买卖、租赁、抵押的法律流程，分析交易中的法律风险，提供合同审查和纠纷解决方案。

## 何时使用
- 用户准备购房/卖房需要了解法律流程时
- 遇到房产交易纠纷需要法律建议时
- 触发词：买房、卖房、房产交易、房屋过户、房产纠纷、购房合同、开发商、二手房

## 工作流程
1. **确认交易类型**：新房买卖/二手房买卖/房产租赁/房产抵押
2. **购房法律问题**：
   - 商品房预售许可证的查验
   - 购房合同的关键条款（面积误差、交房条件、产权办理）
   - 预告登记与网签的作用
   - 按揭贷款的法律安排
3. **二手房交易**：
   - 产权核验（是否被查封/抵押/有共有人）
   - 购房资格和税费计算
   - 资金监管和过户流程
4. **常见纠纷分析**：
   - 开发商逾期交房/无法办理产权证
   - 二手房产权瑕疵
   - 房屋质量问题的维权路径
5. **风险防范**：交易中的常见陷阱和防范措施

## 输出格式
- **交易类型分析**
- **法律流程**：分步说明交易流程
- **关键合同条款**：需要特别注意的条款
- **税费计算**：买卖双方各需承担的税费
- **风险提示**：交易中的常见法律风险
- **纠纷解决路径**
