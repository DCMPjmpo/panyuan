# 竞业限制审查

## 目的
审查竞业限制协议的限制范围、期限、补偿金标准是否符合法律规定，平衡用人单位的商业秘密保护与劳动者的就业权利。

## 何时使用
- 劳动者被要求签署竞业限制协议需要审查时
- 用人单位需要制定竞业限制条款时
- 触发词：竞业限制、竞业禁止、非竞争协议、不竞争条款、限制就业、竞业补偿

## 工作流程
1. **确认适用主体**：是否为高级管理人员、高级技术人员或其他负有保密义务的人员
2. **审查限制范围**：竞争对手的定义是否过于宽泛（应具体列举或划定行业范围）
3. **审查地域范围**：地域限制是否合理（应与用人单位的实际经营范围相关）
4. **审查限制期限**：是否超过法定上限（劳动合同法规定≤2年）
5. **审查补偿金**：月补偿金是否达到法定标准（不低于劳动合同终止前12个月平均工资的30%，且不低于当地最低工资）
6. **审查违约金**：违约金是否合理（应与补偿金存在对应关系）
7. **审查生效条件**：是否约定竞业限制的启动条件和通知义务
8. **出具审查意见**：标注违法或显失公平的条款

## 输出格式
- **竞业限制基本信息**：限制期限、地域、行业范围
- **补偿金计算**：法定最低补偿 vs 约定补偿
- **合规评估**：是否合法有效
- **风险提示**：可能被认定无效的条款
- **修改建议**：兼顾双方利益的条款调整方案
