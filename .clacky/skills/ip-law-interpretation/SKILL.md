# 知识产权法解读

## 目的
解读《专利法》《商标法》《著作权法》的核心规则，包括权利取得、保护范围、侵权认定和救济措施。

## 何时使用
- 用户询问知识产权法律问题时
- 需要了解专利/商标/著作权的保护规则时
- 触发词：专利法、商标法、著作权法、知产法律、IP法律、知识产权法解读

## 工作流程
1. **识别知识产权类型**：专利/商标/著作权/其他
2. **定位法律规则**：
   - 专利法：授予条件（新颖性/创造性/实用性）、保护范围、侵权认定
   - 商标法：显著性要求、混淆可能性判断、驰名商标保护
   - 著作权法：独创性要求、合理使用、法定许可
3. **权利取得与维护**：申请流程、保护期限、续展/年费
4. **侵权判断标准**：全面覆盖原则（专利）/混淆可能性（商标）/实质性相似（著作权）
5. **救济措施**：停止侵权、损害赔偿、诉前禁令
6. **最新法律动态**：专利法/商标法/著作权法的最新修订

## 输出格式
- **权利类型识别**和适用法律
- **法律规则解读**：核心规则的详细分析
- **侵权判断框架**：判断是否侵权的步骤
- **保护建议**：如何保护和行使知识产权
- **最新修法动态**：法律变化对企业/个人的影响
