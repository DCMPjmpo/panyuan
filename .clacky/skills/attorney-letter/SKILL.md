# 律师函

## 目的
生成正式的律师函，用于权利主张、违约催告、侵权警告、催收款项等场景，包含权利依据、法律后果和履行期限。

## 何时使用
- 用户需要向对方发送正式的法律催告时
- 作为诉讼前的最后警告程序时
- 触发词：律师函、发律师函、催告函、法律函、警告函、催收函、发函警告

## 工作流程
1. **确认发函目的**：催告履行/侵权警告/解除合同/主张权利
2. **收集当事人信息**：发函方和收函方的基本信息
3. **梳理事实**：整理纠纷的起因、经过和当前状态
4. **明确权利依据**：列出合同条款或法律条文
5. **提出具体要求**：明确要求对方做什么、在多长时间内完成
6. **说明法律后果**：告知不履行的法律后果（起诉、索赔等）
7. **生成律师函**：按照标准格式生成正式律师函

## 输出格式
按照律师函标准格式：
- **律师事务所抬头**
- **文号**：[年份]X律函字第XX号
- **收函方信息**
- **正文**：事实陈述 → 法律分析 → 权利主张 → 具体要求 → 法律后果
- **履行期限**：明确的截止日期或天数
- **律师事务所盖章/律师签名/日期**
- **联系方式**
