# 强制执行申请

## 目的
根据生效判决书、调解书或仲裁裁决，生成向法院申请强制执行的申请书，附执行依据和财产线索。

## 何时使用
- 对方不履行生效判决/调解书/仲裁裁决时
- 需要申请法院强制执行时
- 触发词：强制执行、申请执行、执行判决、执行书、法院执行、不履行判决、执行立案

## 工作流程
1. **确认执行依据**：判决书/调解书/仲裁裁决书是否已生效
2. **确认申请时效**：检查是否在执行时效内（一般为2年）
3. **梳理执行内容**：明确判决中需要执行的具体内容（金钱给付/行为履行）
4. **收集财产线索**：询问被申请执行人的财产信息（银行账户、房产、车辆等）
5. **计算执行金额**：本金+利息+迟延履行金（日万分之一点七五）+诉讼费
6. **确定执行法院**：一般为第一审法院或财产所在地法院
7. **生成申请书**：按照法院格式生成强制执行申请书

## 输出格式
- **标题**：强制执行申请书
- **申请执行人信息**
- **被执行人信息**
- **执行依据**：生效法律文书的案号和主要内容
- **执行事项**：具体的执行请求
- **财产线索**：已知的被执行人财产信息
- **执行金额计算**：详细的计算明细
- **此致/执行法院**
- **申请人/日期**
