# 消费维权

## 目的
指导消费者在购买商品或接受服务权益受损时，通过投诉、举报、仲裁或诉讼等方式维权，包括退一赔三、退一赔十等惩罚性赔偿的适用。

## 何时使用
- 购买到假冒伪劣商品需要维权时
- 遭遇消费欺诈需要索赔时
- 触发词：消费维权、买到假货、退一赔三、消费者权益、投诉、欺诈、食品安全、虚假宣传

## 工作流程
1. **确认消费纠纷类型**：商品质量问题/虚假宣传/价格欺诈/食品安全/预付式消费
2. **收集证据**：购物凭证、付款记录、商品照片、广告截图、聊天记录
3. **分析法律适用**：
   - 一般消费欺诈：退一赔三（消费者权益保护法第55条）
   - 食品安全问题：退一赔十（食品安全法第148条）
   - 七日无理由退货（网络购物）
4. **选择维权路径**：
   - 与经营者协商
   - 向12315平台投诉
   - 向行业主管部门举报
   - 消费者协会调解
   - 仲裁或诉讼
5. **计算索赔金额**：返还价款+惩罚性赔偿+其他损失

## 输出格式
- **消费纠纷定性**
- **法律适用分析**：适用的法律条款
- **赔偿金额计算**：退款+惩罚性赔偿的详细计算
- **证据清单**：需要收集的证据
- **维权路径**按优先级排列
- **投诉模板**：投诉信的写作要点
