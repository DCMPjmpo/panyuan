# 案例检索分析

## 目的
根据案件事实和争议焦点，检索类似案例，分析法院的裁判倾向、胜诉率、赔偿标准和争议焦点分布，为诉讼决策提供数据支持。

## 何时使用
- 需要了解类似案件的裁判结果时
- 需要评估诉讼胜率和赔偿预期时
- 触发词：案例检索、类似案例、判例、裁判倾向、胜诉率、司法观点、有没有类似案例

## 工作流程
1. **确定检索要素**：案由、争议焦点、关键事实要素、法律依据
2. **确定检索范围**：审理法院层级、地域、时间范围
3. **分析裁判趋势**：
   - 支持/驳回/部分支持的占比
   - 平均赔偿/支持金额
   - 不同地区法院的裁判差异
4. **提取裁判观点**：法院对类似争议焦点的典型论述
5. **评估案件风险**：基于大数据预测本方案件的走向
6. **提供策略建议**：根据案例分析结果建议诉讼或调解策略

## 输出格式
- **检索条件**：案由、关键词、法院层级、时间范围
- **检索结果概览**：检索到X件相关案例
- **裁判统计**：
  - 支持率/驳回率/部分支持率
  - 平均赔偿金额
  - 地区分布
- **典型案例摘要**：选取3-5个最相似的案例
- **司法观点总结**：法院的典型裁判观点
- **风险评估**：基于数据分析的胜诉率和预期赔偿
- **策略建议**：根据案例分析的诉讼策略
