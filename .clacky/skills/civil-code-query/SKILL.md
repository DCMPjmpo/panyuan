# 民法典查询

## 目的
根据用户的法律问题，查询《中华人民共和国民法典》的相关条文，解读条文的适用条件、法律后果和司法实践中的理解。

## 何时使用
- 用户询问民法相关问题时
- 需要了解民法典某一条文的具体含义时
- 触发词：民法典、民法、物权、债权、人格权、婚姻家庭、继承、侵权、合同编、物权编

## 工作流程
1. **识别问题领域**：确认属于民法典七编中的哪一编
2. **精确定位条文**：根据问题定位到具体的编、章、条
3. **解读条文**：
   - 字面含义
   - 立法目的和背景
   - 适用条件和范围
   - 与其他条文的关系
4. **司法解释关联**：关联相关的最高人民法院司法解释
5. **典型案例参考**：引用相关案例帮助理解
6. **实务建议**：基于条文的实务操作建议

## 输出格式
- **条文原文**：引用具体的民法典条文
- **条文解读**：
  - 适用范围
  - 构成要件
  - 法律效果
  - 与其他条文的关系
- **相关司法解释**：关联的司法解释条文
- **典型案例**：帮助理解的案例
- **实务提示**：操作中的注意事项
