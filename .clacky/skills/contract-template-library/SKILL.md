# 合同模板库

## 目的
提供各类常用合同的标准化模板，包含填写说明、可选条款和法律提示，帮助用户快速起草合法合规的合同。

## 何时使用
- 用户需要起草一份合同但不知道格式时
- 用户需要参考标准合同模板时
- 触发词：合同模板、合同范本、合同样本、模板下载、标准合同、合同怎么写

## 工作流程
1. **确认合同类型**：询问用户需要什么类型的合同
2. **了解关键信息**：签约主体、核心交易条件
3. **提供合同模板**：提供该类型合同的标准模板
4. **标注填空位置**：明确标注需要用户填写的内容
5. **可选条款提示**：标注可选条款及选择条件
6. **法律提示**：标注需要特别注意的条款和相关法律规定

## 输出格式
- **合同模板名称**
- **适用场景说明**
- **合同正文模板**：[填空内容用方括号标注]
- **填写说明**：各项填空内容的要求
- **可选条款**：可根据实际情况增删的条款
- **重要法律提示**：需要特别注意的条款和法律依据
- **签约注意事项**

**支持的合同类型包括但不限于**：
- 买卖合同、租赁合同、借款合同、赠与合同
- 服务合同、承揽合同、委托合同
- 合伙协议、股东协议、股权转让协议
- 保密协议、竞业限制协议
- 劳动合同、劳务合同、聘用合同
- 和解协议、调解协议
